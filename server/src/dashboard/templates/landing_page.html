How to Connect?

Dashboards are the front door. The API is the full archive.

The dashboards help visitors discover datasets, while the query API endpoint give access to the complete data available in the database.

Through dashboards Each dashboard ships with a download button. Visitors connect a wallet, modify a pre-configured query, if necessary sign the x402 payment in-browser, and the resulted data is converted to a CSV file — no code required.
Through code Send a POST /api/query with a simple SELECT — WHERE, ORDER BY, and LIMIT are supported; Subqueries, JOIN, CTEs, GROUP BY, aggregates, and window functions are rejected (see the docs for the full grammar). Wrap the call using any x402 Protocol client library and it handles pricing, settlement, and signing for you. Results stream back as Apache Arrow IPC.
